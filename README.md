## Description

**Vibas** allows for visual designing of flowchart algorithms, their simulation, and analysis. This makes it easy to test and present various algorithmic solutions without the need to write code. **Vibas** is a desktop application written in .NET 10, serving as a flowchart algorithm simulator. The project enables creating, editing, and running flowcharts in a user-friendly graphical environment. It is also possible to write algorithms in code (in C#), which are automatically converted into corresponding blocks. This way, you can quickly visualize your code and analyze its behavior more easily.

---

## Features

- Creating and editing flowcharts
- Step-by-step algorithm simulation
- Intuitive graphical interface
- Saving and loading projects

## Installation and Running

1. Clone the repository
   - In the console: `git clone https://github.com/anycodev2/vibas.git`
   - Or in GitHub Desktop: clone repository -> URL `https://github.com/anycodev2/vibas.git`
2. Open the project in Visual Studio 2026
3. Press F5 to run the application
